python中 r'', b'', u'', f'' 的含义

r/R:非转义的原始字符

r'input\n'

在此\n不为换行，为两个字符

b:bytes

b'input\n'  # bytes字节符，打印以b开头。

u/U:表示unicode字符串

u'input\n' # unicode编码字符，python3默认字符串编码方式

f/format():格式化操作

print(f'{name} done in {time.time() - t0:.2f} s')  # 以f开头表示在字符串内支持大括号内的python 表达式
